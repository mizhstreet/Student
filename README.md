# 学校用の学生管理アプリ
# 機能
- 学生管理 
- クラス管理
- 科目管理
- 試験、点数管理
- 出席管理
# Usage
- SQL Serverでcsdl.txt実行（1行ずつ）
- DBUtility.java を開いて、ポートとデータベースのユーザー名, パスワードを修正してください。
- ユーザー名はadmin,パスワードはadmin、ログインしてください
- 私のプロジェクトを使いたい場合はクレジットを挿入することを忘れないで下さい!
